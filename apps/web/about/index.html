About Livefield

Livefield is an unofficial Twitch-focused observation tool built to understand how viewership distributes, shifts, and overlaps in real time.

What this is

Livefield visualizes Twitch activity through three perspectives: current distribution, daily flow, and rivalries between streams.

What you can see

  • Heatmap: current viewer distribution across streams
  • Day Flow: how viewership moves over time
  • Rivalry Radar: overlap and competition between streams

How to read

  • Viewers are primary: all charts are driven by viewer signals
  • Activity is secondary: missing chat does not invalidate charts
  • Status: live / partial / stale / empty describe data coverage

Data limitations

This project prioritizes transparency over completeness. "Partial" or "stale" does not mean broken — it means observed coverage is incomplete.

Scope

This is a Twitch-only MVP focused on stable observation rather than perfect coverage.
